# THE TO DO LIST APP 

#### Date, 2024/04/12

#### By *Gloria Wekura*

## Description
This is an app the the user is able to input the tasks that he/ she is required to complete and once they are done they can double click to mark as done or delete them. They can also edit the tasks.
